# Planner

A very simple planner to keep track of your routines/todos/notes. Stores data only in your browser.

## Demo -> [Planner](https://websymphony.github.io/planner)

### Built using:

* React
* Mobx
* localforage
* styled-components
* rebass
* @reach/router

### To run on your machine:

1. Download the repo.
2. Run `yarn install`
3. Then run `yarn start`
